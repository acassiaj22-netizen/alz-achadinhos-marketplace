# ALZ Achadinhos Marketplace

Projeto desenvolvido estritamente como um site estático (HTML/CSS) focado em alta conversão e mobile-first, ideal para links de afiliados.

## Estrutura de Pastas:
- `/marketplace` (Isolamento por plataforma)
  - `/shopee`
  - `/mercadolivre`
  - `/magalu`
  - `/kiwify`
- `/pages` (Páginas principais e templates)
- `/assets` (Imagens e Logos)
- `/links` (Organização de URLs)
- `/shared` (Estilos e componentes globais)

## Deploy
Este projeto é implantado na **Vercel**. Como não há backend, basta configurar o preset como `Other` e fazer o deploy automático pelo GitHub.
